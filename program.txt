STORE 0 5     ; Сохраняем 8 в память по адресу 5
LOAD 1 2      ; Загружаем 2 в регистр 1
LOAD 0 5      ; Загружаем значение из памяти по адресу 5 в регистр 0
ADD 0 1       ; Складываем регистры 0 и 1, результат в регистре 0 (теперь это 10)
STORE 0 6     ; Сохраняем результат из регистра 0 в память по адресу 6
